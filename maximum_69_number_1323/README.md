# 1323. Maximum 69 Number

## Descrição

Dado um número inteiro positivo `num` consistindo apenas dos dígitos 6 e 9, retorne o número máximo que você pode obter alterando no máximo um dígito (6 se torna 9, e 9 se torna 6).

**Exemplo 1:**

```
Input: num = 9669
Output: 9969
```

Explicação: Alterar o primeiro dígito resulta em 6669. Alterar o segundo dígito resulta em 9969. Alterar o terceiro dígito resulta em 9699. Alterar o quarto dígito resulta em 9666. O número máximo é 9969.

**Exemplo 2:**

```
Input: num = 9996
Output: 9999
```

Explicação: Alterar o último dígito 6 para 9 resulta no número máximo.

**Exemplo 3:**

```
Input: num = 9999
Output: 9999
```

Explicação: É melhor não aplicar nenhuma alteração.

**Restrições:**

- `1 <= num <= 10^4`
- Os dígitos de `num` são 6 ou 9.
