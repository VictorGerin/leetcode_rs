# 524. Longest Word in Dictionary through Deleting

## Descrição

Dada uma string `s` e uma lista de strings `dictionary`, a tarefa é encontrar a string mais longa no dicionário que pode ser formada deletando alguns caracteres da string dada `s`. Se houver múltiplos resultados possíveis, retorne a palavra mais longa com a menor ordem lexicográfica. Se nenhuma palavra existir, retorne uma string vazia.

**Exemplo 1:**

```
Input: s = "abpcplea", dictionary = ["ale","apple","monkey","plea"]
Output: "apple"
```

**Exemplo 2:**

```
Input: s = "abpcplea", dictionary = ["a","b","c"]
Output: "a"
```

**Restrições:**

1. Todas as strings na entrada consistem apenas de letras minúsculas do inglês.
2. O tamanho do dicionário não excederá 1.000.
3. O comprimento de todas as strings na entrada não excederá 1.000.
