# 287. Find the Duplicate Number

## Descrição

Dado um array de inteiros `nums` contendo `n + 1` inteiros onde cada inteiro está no intervalo `[1, n]` inclusive, existe apenas um número repetido em `nums`. Sua tarefa é encontrar esse número repetido.

**Restrições:**

- Você não deve modificar o array (assuma que o array é somente leitura).
- Você deve usar apenas espaço extra constante, O(1).
- Sua complexidade de tempo deve ser menor que O(n²).
- Existe apenas um número duplicado no array, mas ele pode ser repetido mais de uma vez.

**Exemplo 1:**

```
Input: nums = [1,3,4,2,2]
Output: 2
```

**Exemplo 2:**

```
Input: nums = [3,1,3,4,2]
Output: 3
```
