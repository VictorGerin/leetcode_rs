# 67. Add Binary

## Descrição

Dadas duas strings binárias `a` e `b`, retorne sua soma como uma string binária.

**Exemplo 1:**

```
Input: a = "11", b = "1"
Output: "100"
```

**Exemplo 2:**

```
Input: a = "1010", b = "1011"
Output: "10101"
```

**Restrições:**

- `1 <= a.length, b.length <= 10^4`
- `a` e `b` consistem apenas dos caracteres `'0'` ou `'1'`.
- Cada string não contém zeros à esquerda, exceto para o próprio zero.
